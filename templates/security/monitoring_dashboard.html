خطوات التثبيت:
  1. تثبيت Docker (إذا لم يكن مثبت)
  2. تشغيل Prometheus: 
    docker run -d -p 9090:9090 prom/prometheus
  3. تشغيل Grafana: 
    docker run -d -p 3000:3000 grafana/grafana
  4. فتح Grafana على http://localhost:3000
  5. تسجيل الدخول: admin/admin
  6. إضافة Prometheus كمصدر بيانات
  7. استيراد Dashboard للمراقبة
