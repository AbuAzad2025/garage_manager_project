🛑 إيقاف النظام
════════════════════════════════════════════════════════════════════════════

الطريقة الأولى:
اضغط Ctrl + C مرتين في نافذة PowerShell

الطريقة الثانية (إيقاف إجباري):

الطريقة الثالثة (إيقاف منفذ معين):

⚠️ حل المشاكل الشائعة
════════════════════════════════════════════════════════════════════════════

❌ المشكلة: cannot be loaded because running scripts is disabled
✅ الحل:
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser


❌ المشكلة: Port 5000 already in use
✅ الحل:
   Get-Process -Id (Get-NetTCPConnection -LocalPort 5000).OwningProcess | Stop-Process -Force


❌ المشكلة: ModuleNotFoundError
✅ الحل:
   .\.venv\Scripts\Activate.ps1
   pip install -r requirements.txt


❌ المشكلة: Database not found
✅ الحل:
   .\.venv\Scripts\Activate.ps1
   python -c "from app import create_app, db; app = create_app(); app.app_context().push(); db.create_all()"
