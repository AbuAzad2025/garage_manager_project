# نظام إدارة الكراج (Garage Manager)

نظام متكامل لإدارة الكراجات والورشات مع محاسبة متقدمة ومساعد ذكي.

## المتطلبات

- Python 3.8+
- SQLite أو PostgreSQL

## التثبيت

```bash
pip install -r requirements.txt
```

## إعداد النظام

```bash
flask db upgrade
flask seed-all --allow-default-passwords
```

## تشغيل النظام

```bash
python app.py
```

أو على Windows:
```powershell
.\RUN.ps1
```

أو على Linux/Mac:
```bash
./run.sh
```

## المميزات

- إدارة العملاء والموردين والشركاء
- إدارة المخزون والمستودعات
- نظام المبيعات والفواتير
- إدارة الخدمات وطلبات الصيانة
- نظام المصاريف والمدفوعات
- محاسبة متقدمة (دفتر الأستاذ، التسويات البنكية، مراكز التكلفة)
- نظام الشحنات والاستيراد
- تقارير مالية شاملة
- نظام صلاحيات متقدم
- مساعد ذكي (AI Assistant)
- إشعارات في الوقت الفعلي
- دعم متعدد العملات
